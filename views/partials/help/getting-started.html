Importing Libraries from EndNote

  1. Create an account in the SRA or login if you already have one
  2. First export your library from Endnote selecting "show all fields" and saving the file as xml.
  3. Import your library into the SRA by selecting "Import Library" from the tools menu or from the sidebar on the left of the screen
  4. Click on your library to view it or select the button to the left of its listing to perform an operation

Exporting libraries back into EndNote

  1. Use the "Tools" drop down menu on the top right hand side of the screen.
  2. Select export references
  3. Choose "Endnote XML"
  4. Select "Download"
